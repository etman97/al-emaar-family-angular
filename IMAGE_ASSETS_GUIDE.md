# EMMAR - Image Assets Guide

## Required Images

Please add the following images to the `public/assets/` folder:

### 1. Logo
- **Filename**: `logo.png`
- **Location**: `public/assets/logo.png`
- **Usage**: Navbar logo

### 2. Building Images for Home Page
You need **3 building images**:

#### Main Building (Largest, center-left)
- **Filename**: `building-main.jpg`
- **Location**: `public/assets/building-main.jpg`
- **Recommended size**: 600x550px
- **Position**: Center-left, largest image

#### Top Building (Smaller, top-left)
- **Filename**: `building-top.jpg`
- **Location**: `public/assets/building-top.jpg`
- **Recommended size**: 450x320px
- **Position**: Top-left, overlapping the main image

#### Bottom Building (Smaller, bottom-right)
- **Filename**: `building-bottom.jpg`
- **Location**: `public/assets/building-bottom.jpg`
- **Recommended size**: 450x280px
- **Position**: Bottom-right

## How to Add Images

1. Create the `assets` folder if it doesn't exist:
   ```
   public/assets/
   ```

2. Add your images with the exact filenames mentioned above

3. The layout will automatically display them in the correct positions

## Design Colors Used
